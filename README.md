Function.prototype.bind Polyfill for Meteor
===========================================

This polyfill is for PhantomJS on Travis.

Use in App
----------
```Bash
meteor add zodiase:function-bind
```

Use in Package
--------------
In package.js:
```JavaScript
api.use('zodiase:function-bind@0.0.1');
```
